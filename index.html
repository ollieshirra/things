Vector Artwork Explained

Overview

There are two categories which all artwork and graphics fit in to, these are vector and bitmap. This applies to anything you view on a screen whether it be a logo on an app or a intricate illustration on a website.

Vector artwork is made using software such as Adobe Illustrator and consists of mathematical calculations which define shapes, colours, positioning etc. Bitmap artwork is made of pixels, the small squares of individual colour which together form the full image.

Designers love vectors as they are easy to edit and manipulate. They can also be scaled up to any size without loosing quality.

How To Tell The Difference

The easiest way to tell between the two is to zoom-in to the artwork and see if you can see any pixels. If you have zoomed in as far as possible and everything is still sharp then its vector. If you can see pixels then its bitmap. Some artwork does combine both types however so be sure to look over the full artwork.

Terminology

Vector is the most used term with the variation of ‘vectored’. Another word which may be used is ‘paths’ as vector objects are made up of lines know as ‘paths’.

Bitmap artwork may also be referred to as ‘raster’ or ‘rasterised’. To rasterise is a process to turning vector artwork into bitmap artwork, however this cannot be done the other way around.

Real Life Examples

Below is a list of image types and file formats you may have come across:

Photograph

Always a bitmap.

Artwork

Could be vector, bitmap or a mixture of both.

Graphics

Could be vector, bitmap or a mixture of both.

Logo

Could be vector, bitmap or a mixture of both.

Scan

Anything you scan in to a computer will be made in to a bitmap file.

.jpg

Always a bitmap.

.png

Always a bitmap.

.ai

This file extension stands for Adobe Illustrator which is software for creating and editing vectors. This usually means the artwork in the file is vector, however it is possible to save bitmap artwork in an AI file so the artwork may be bitmap, vector or a mixture of the two.

.eps

This is a common vector format which can be opened in various software. The same applies to this format as above. The artwork is usually vector, however it may be fully bitmap, fully vector or a mixture of the two.

.pdf

PDF files can be fully vector, fully bitmap or a mixture of both. Many PDFs are made up of only editable text which is inherently vector. However PDFs can contain photos along with text and vector graphics resulting in a mixture. Or PDFs can contain only vector elements.

.psd

PSD is the native format for Adobe Photoshop files. These files are almost always bitmap however some Photoshop files can have vector artwork embedded which will open in vector editing software when the layer is double clicked.
